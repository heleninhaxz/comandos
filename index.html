Comandos Do Cmd Comandos do linux

Cmd

Conceito: o prompt de comando(cmd) é um programa que emula o campo de entrada em uma tela de interface de usuário baseada em texto com a interface gráfica do usuário (IU) do Windows.

Comandos

  • Cd: (alterar diretório) Comando usado para alternar o diretório (caminho) no qual você está trabalhando.

  • Dir: (Diretório) Permite visualizar arquivos e pastas no diretório selecionado. Fornece também detalhes sobre a data de modificação e o tamanho dos arquivos.

  • Systeminfo:

  • Ipconfig: Exibe as configurações de IP para todas as conexões de rede.

  • mkdir: Este comando cria um novo diretório.

  • type: Esse comando serve para exibir o conteúdo dentro de um arquivo, seja ele um arquivo Word ou txt.

  • copy: Copia arquivos.

  • ping: Testa a conectividade com um host específico na rede.

  • echo: Exibe mensagens de texto ou ativa ou desativa o eco de comandos.

  • attrib: Exibe ou altera os atributos de um arquivo ou diretório.

  • rmdir: Remove um diretório

  • move: Move arquivos e renomeia diretórios.

  • del: Exclui arquivos.

  • hostname: Exibe o nome do computador atual.

  • title: Define o título da janela do prompt de comando.

  • tree: Exibe uma representação gráfica da estrutura de diretórios de uma unidade ou caminho especificado.

  • help: exibe uma lista de comandos internos disponíveis e fornece uma breve descrição de cada um.

  • subst: Associa um caminho de diretório a uma letra de unidade.

  • whoami: Exibe o nome de usuário e o grupo de segurança do usuário atual.

  • xcopy: Copia diretórios e arquivos, oferecendo mais opções que o comando copy.
